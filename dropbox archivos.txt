✅ PASO 3 – LECTURA DEL CONTENIDO
Proceso:
Utilizar herramienta interna o librería Python como PyMuPDF (fitz) o pdfplumber para abrir y extraer texto.

Leer únicamente las primeras 5–10 páginas para un resumen inicial.

Extraer texto limpio, evitando encabezados, números de página, notas editoriales, márgenes.

✅ PASO 4 – ANÁLISIS DE CONTENIDO
Puntos clave:
Identificar tema central, tono, estructura narrativa y estilo de redacción.

Clasificar el contenido según categorías RAPEALO: historia, crítica cultural, elementos del Hip Hop, biografía, filosofía.

Resumir en bloques temáticos de 3 a 5 ideas clave.

✅ PASO 5 – REDACCIÓN DEL RESUMEN
Estilo:
Narrativo, directo, respetuoso del autor original.

Evitar adornos, palabras abstractas o exageraciones.

No utilizar academicismo vacío ni lenguaje técnico innecesario.

import pandas as pd
import fitz  # PyMuPDF

# Leer índice
df = pd.read_excel("Indice_RAPEALO.xlsx")
libros = df[df.apply(lambda row: row.astype(str).str.contains("libro", case=False).any(), axis=1)]

# Obtener archivo fuente
archivo = libros.iloc[0]['archivo_fuente']

# Abrir PDF
doc = fitz.open(f"/ruta/{archivo}")
texto = ""
for page in doc[:5]:
    texto += page.get_text()

# Procesar texto en bloques
bloques = texto.split("\n\n")[:4]  # Simplificado
